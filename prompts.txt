We are implementing a major improvement to the assessment system’s academic and permission structure.

New Subject-Based Teaching and Access Model
1. Multi-Subject Teacher Support

A teacher can now teach one or multiple subjects across different classes and year groups.

Examples:

A teacher may teach:
ICT and Core Mathematics
Physics and Elective Mathematics
English Language in multiple classes

The system must support:

many-to-many relationship between teachers and subjects
many-to-many relationship between teachers and classes
2. Subject-Based Question Bank

The question bank must now be organized by subject.

Every question must belong to a specific subject.

Examples:

ICT questions → visible to ICT teachers
Mathematics questions → visible to Mathematics teachers
Physics questions → visible to Physics teachers

Questions should no longer be isolated to only the teacher who created them.

3. Shared Subject Collaboration

Teachers teaching the same subject should collaborate through a shared question bank and shared tests.

Example:

All ICT teachers can:
view ICT questions
use ICT questions
edit permitted ICT questions
create ICT tests
reuse ICT assessments

Another example:

A Mathematics teacher should NOT access ICT questions unless assigned to ICT.
4. Subject-Level Access Control

Access permissions should be determined primarily by subject assignment.

A teacher should only access:

subjects assigned to them
questions for those subjects
tests created under those subjects

Example:

Teacher A teaches:
- ICT
- Computing

Teacher A can access:
- all ICT questions
- all Computing questions
- ICT tests
- Computing tests
5. Shared Test Visibility

Tests and examinations should also become subject-based.

Example:

All ICT teachers can see:
ICT quizzes
ICT tests
ICT exams

This allows:

collaboration
standardization
moderation
easier assessment preparation
6. Ownership and Audit Tracking

Even though resources are shared, the system must still track:

who created the question
who edited it
when it was modified

Add fields such as:

created_by
updated_by
created_at
updated_at
7. Recommended Permission Logic
Teachers CAN:
access all questions for subjects they teach
reuse questions from other teachers within the same subject
collaborate on assessments
Teachers CANNOT:
access questions for subjects they are not assigned to
modify restricted or locked assessments
access administrative controls unless permitted
