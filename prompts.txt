2. Strands
CREATE TABLE subject_strands (
    id INT UNSIGNED AUTO_INCREMENT PRIMARY KEY,

    school_id INT UNSIGNED NOT NULL,

    subject_id INT UNSIGNED NOT NULL,

    strand_code VARCHAR(50) NOT NULL,

    strand_label VARCHAR(255) NOT NULL,

    description TEXT NULL,

    sort_order SMALLINT UNSIGNED DEFAULT 0,

    is_active TINYINT(1) DEFAULT 1,

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    FOREIGN KEY (subject_id)
        REFERENCES subjects(id)
        ON DELETE CASCADE
);

Example:

ICT
 ├─ Computer Systems
 ├─ Networking
 ├─ Data Security
 └─ Programming
3. Sub-Strands
CREATE TABLE subject_sub_strands (
    id INT UNSIGNED AUTO_INCREMENT PRIMARY KEY,

    strand_id INT UNSIGNED NOT NULL,

    sub_strand_code VARCHAR(50),

    sub_strand_label VARCHAR(255) NOT NULL,

    description TEXT NULL,

    sort_order SMALLINT UNSIGNED DEFAULT 0,

    is_active TINYINT(1) DEFAULT 1,

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    FOREIGN KEY (strand_id)
        REFERENCES subject_strands(id)
        ON DELETE CASCADE
);

Example:

Networking
 ├─ Network Types
 ├─ Transmission Media
 ├─ Internet Services
 └─ Wireless Communication
4. Topics

This is where most systems stop too early. You should definitely have a Topics table.

CREATE TABLE subject_topics (
    id INT UNSIGNED AUTO_INCREMENT PRIMARY KEY,

    sub_strand_id INT UNSIGNED NOT NULL,

    topic_code VARCHAR(50),

    topic_label VARCHAR(255) NOT NULL,

    description TEXT NULL,

    sort_order SMALLINT UNSIGNED DEFAULT 0,

    is_active TINYINT(1) DEFAULT 1,

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    FOREIGN KEY (sub_strand_id)
        REFERENCES subject_sub_strands(id)
        ON DELETE CASCADE
);

Example:

Transmission Media
 ├─ Twisted Pair Cable
 ├─ Coaxial Cable
 ├─ Fibre Optic Cable
 ├─ Guided Media
 └─ Unguided Media
5. Questions

Instead of storing only text labels, store the IDs.

CREATE TABLE questions (
    id BIGINT UNSIGNED AUTO_INCREMENT PRIMARY KEY,

    school_id INT UNSIGNED NOT NULL,

    subject_id INT UNSIGNED NOT NULL,

    strand_id INT UNSIGNED NOT NULL,

    sub_strand_id INT UNSIGNED NOT NULL,

    topic_id INT UNSIGNED NULL,

    author_id INT UNSIGNED NOT NULL,

    type VARCHAR(50) NOT NULL,

    bloom_level TINYINT DEFAULT 1,

    difficulty ENUM('easy','medium','hard'),

    year_group TINYINT NOT NULL,

    marks DECIMAL(5,2) DEFAULT 1,

    question_text LONGTEXT NOT NULL,

    explanation LONGTEXT NULL,

    is_active TINYINT(1) DEFAULT 1,

    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,

    FOREIGN KEY (subject_id)
        REFERENCES subjects(id),

    FOREIGN KEY (strand_id)
        REFERENCES subject_strands(id),

    FOREIGN KEY (sub_strand_id)
        REFERENCES subject_sub_strands(id),

    FOREIGN KEY (topic_id)
        REFERENCES subject_topics(id)
);
